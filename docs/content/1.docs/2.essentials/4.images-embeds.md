## 图片

### 使用 Markdown

使用标准 Markdown 语法显示图片。Markdown 图片对于基本图片显示来说简单易用。

::code-preview
![Nuxt Social Image](https://nuxt.com/new-social.jpg)

#code
```mdc
![Nuxt Social Image](https://nuxt.com/new-social.jpg)
```
::

::note
如果安装了 [`@nuxt/image`](https://image.nuxt.com/get-started/installation)，则会使用 `<NuxtImg>` 组件代替原生的 `img` 标签。
::

### 使用嵌入

使用嵌入以实现更多图片自定义。嵌入提供了对图片属性（如大小和样式）的更精细控制。

::code-preview
![Nuxt Social Image](https://nuxt.com/new-social.jpg){height="150"}



#code
```mdc
:img{src="https://nuxt.com/new-social.jpg" alt="Nuxt Social Image" height="150"}
```
::
