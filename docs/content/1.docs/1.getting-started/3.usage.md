## 编写内容

你可以直接在 [`content/`](https://content.nuxt.com/usage/content-directory) 目录中开始编写 `.md` 或 `.yml` 文件，页面会自动更新。左侧栏与移动端菜单的导航会根据内容结构自动生成，同时你也可以通过全文搜索快速浏览内容。
