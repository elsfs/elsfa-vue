## 快速开始

你可以使用以下命令快速创建一个全新的项目：

```bash [Terminal]
npx nuxi init -t github:nuxt-ui-templates/saas
```

或者在 GitHub 上从模板创建一个新仓库：

1. 打开 <https://github.com/nuxt-ui-templates/saas>
2. 点击 `Use this template` 按钮
3. 输入仓库名称，并点击 `Create repository from template` 按钮
4. 克隆你新创建的仓库
5. 使用你喜欢的包管理器安装依赖
6. 启动开发服务器

就是这样！现在你可以在 [`content/`](https://content.nuxt.com/usage/content-directory) 目录中开始编写文档了 🚀
