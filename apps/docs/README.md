# Nuxt SaaS 模板

[![Nuxt UI](https://img.shields.io/badge/%E5%9F%BA%E4%BA%8E-Nuxt%20UI-00DC82?logo=nuxt&labelColor=020420)](https://ui.nuxt.com)

一个功能完备的 SaaS 应用模板，包含着陆页、定价页、文档与博客，全部由 [Nuxt UI](https://ui.nuxt.com) 组件驱动，帮助你快速启动下一个项目。

- [在线演示](https://saas-template.nuxt.dev/)

## 快速开始

```bash [Terminal]
npm create nuxt@latest -- -t ui/saas
```

## 一键部署

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-name=saas&repository-url=https%3A%2F%2Fgithub.com%2Fnuxt-ui-templates%2Fsaas&demo-image=https%3A%2F%2Fui.nuxt.com%2Fassets%2Ftemplates%2Fnuxt%2Fsaas-dark.png&demo-url=https%3A%2F%2Fsaas-template.nuxt.dev%2F&demo-title=Nuxt%20SaaS%20Template&demo-description=A%20SaaS%20template%20with%20landing%2C%20pricing%2C%20docs%20and%20blog%20powered%20by%20Nuxt%20Content.)

## 安装依赖

请先安装依赖：

```bash
pnpm install
```

## 开发服务器

在 `http://localhost:3000` 启动开发服务器：

```bash
pnpm dev
```

## 生产构建

为生产环境构建应用：

```bash
pnpm build
```

本地预览生产构建：

```bash
pnpm preview
```

更多信息请查看 [部署文档](https://nuxt.com/docs/getting-started/deployment)。

## Renovate 集成

在你的仓库中安装 [Renovate GitHub 应用](https://github.com/apps/renovate/installations/select_target) 即可自动化依赖更新。
