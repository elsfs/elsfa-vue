## 标题

使用标题来引入主要章节。它们可以为文档建立结构，帮助用户更好地导航内容。

::code-preview
---
class: "[&>div]:*:my-0"
---
## 标题

#code
```mdc
## 标题
```
::

### 副标题

使用副标题进一步划分章节。它们可以建立更细致的内容层级，使文档更易阅读。

::code-preview
---
class: "[&>div]:*:my-0"
---
### 副标题

#code
```mdc
### 副标题
```
::

::tip
每个标题与副标题都会生成一个锚点，并自动出现在目录中。
::

## 文本格式

Nuxt UI 支持大多数 Markdown 格式选项。

| 样式  | 用法           | 效果         |
| ----- | -------------- | ------------ |
| 粗体  | `**加粗**`     | **加粗**     |
| 斜体  | `*斜体*`       | *斜体*       |
| 删除线 | `~~删除线~~`   | ~~删除线~~   |

可以组合多种格式以获得更丰富的文本样式与视觉强调。

| 样式           | 用法                   | 效果               |
| -------------- | ---------------------- | ------------------ |
| 粗斜体         | `**_bold italic_**`    | ***Bold Italic***  |
| 粗体 + 删除线  | `~~**bold**~~`         | ~~**Bold**~~       |
| 斜体 + 删除线  | `~~*italic*~~`         | ~~*Italic*~~       |

对于上标、下标或数学标注，可使用 HTML 的 `<sup>` 与 `<sub>` 标签。

| 样式     | 用法                         | 效果       |
| -------- | ---------------------------- | ---------- |
| 上标     | `<sup>superscript</sup>`     | superscript|
| 下标     | `<sub>subscript</sub>`       | subscript  |

## 链接

链接用于连接文档的不同部分和外部资源，是用户导航与引用的关键。
要创建一个链接，请将链接文本包裹在 `[]()` 中。

::code-preview
---
class: "[&>div]:*:my-0"
---
[Nuxt UI](https://ui.nuxt.com/getting-started/installation)

#code
```mdc
[Nuxt UI](https://ui.nuxt.com/getting-started/installation)
```
::

### 内部链接

在文档内部进行链接时，使用以根路径开头的相对路径，例如 `/getting-started/installation`。

::code-preview
---
class: "[&>div]:*:my-0"
---
[安装](/getting-started/installation)

#code
```mdc
[安装](/getting-started/installation)
```
::

## 列表

以结构化、易读的方式组织关联项。Markdown 支持无序列表、有序列表与嵌套列表，以满足不同的内容需求。

### 无序列表

当列出的项目没有特定顺序时，请使用无序列表。每一项以 `-` 开头。

::code-preview
---
class: "[&>div]:*:my-0"
---
- 我是列表项。
- 我是另一个列表项。
- 我是最后一个列表项。

#code
```mdc
- 我是列表项。
- 我是另一个列表项。
- 我是最后一个列表项。
```
::

### 有序列表

当项目的顺序很重要（例如操作步骤）时，请使用有序列表。每一项以数字开头。

::code-preview
---
class: "[&>div]:*:my-0"
---
1. 我是列表项。
2. 我是另一个列表项。
3. 我是最后一个列表项。

#code
```mdc
1. 我是列表项。
2. 我是另一个列表项。
3. 我是最后一个列表项。
```
::

### 嵌套列表

通过子项创建分层的列表来表达更复杂的结构。将子项缩进 4 个空格以实现嵌套。

::code-preview
---
class: "[&>div]:*:my-0"
---
- 我是列表项。
  - 我是嵌套的列表项。
  - 我是另一个嵌套的列表项。
- 我是另一个列表项。

#code
```mdc
- 我是列表项。
  - 我是嵌套的列表项。
  - 我是另一个嵌套的列表项。
- 我是另一个列表项。
```
::

## 表格

以清晰的行与列展示结构化数据。表格非常适合用于对比数据或列出属性。

::code-preview
---
class: "[&>div]:*:my-0 [&>div]:*:w-full"
---
| 属性    | 默认值     | 类型     |
| ------- | --------- | -------- |
| `name`  |           | `string` |
| `size`  | `md`      | `string` |
| `color` | `neutral` | `string` |

#code
```mdc
| 属性    | 默认值     | 类型                     |
|---------|-----------|--------------------------|
| `name`  |           | `string`{lang="ts-type"} |
| `size`  | `md`      | `string`{lang="ts-type"} |
| `color` | `neutral` | `string`{lang="ts-type"} |
```
::

## 引用

用于突出显示重要的引文、引用或需要强调的文本。引用可以在视觉上将被引用的内容与正文区分开来。

### 单行引用

单行引用适用于简短且有冲击力的引语或注释。要创建单行引用，在段落前添加一个 `>` 符号即可。

::code-preview
---
class: "[&>div]:*:my-0"
---
> Nuxt UI 是一组用于创建精美且易用用户界面的 Vue 组件、可组合函数与工具集。

#code
```mdc
> Nuxt UI is a collection of Vue components, composables and utils designed to create beautiful and accessible user interfaces.
```
::

### 多行引用

多行引用适用于更长的引文，或需要在同一次引用中包含多个段落的情况。

::code-preview
---
class: "[&>div]:*:my-0"
---
> Nuxt UI is a collection of Vue components, composables and utils designed to create beautiful and accessible user interfaces.
>
> Create beautiful, responsive, and accessible Vue applications with Nuxt UI.

#code
```mdc
> Nuxt UI is a collection of Vue components, composables and utils designed to create beautiful and accessible user interfaces.
>
> Create beautiful, responsive, and accessible Vue applications with Nuxt UI.
```
::
