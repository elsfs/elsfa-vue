该模板基于 [Nuxt UI](https://ui.nuxt.com) 构建，是一个开箱即用的 SaaS 模板，帮助你在数分钟内创建美观且响应迅速的 Nuxt 应用。

它内置了可自定义的着陆页、定价页、文档、博客以及认证页面（登录、注册）。

## 关键特性

该模板提供了一系列功能，旨在简化文档管理：

- 基于 [Nuxt](https://nuxt.com)：采用最新的 Nuxt 框架，性能优异。
- 构建于 [Nuxt UI](https://ui.nuxt.com)：集成完善的 UI 组件套件。
- 通过 [Nuxt Content](https://content.nuxt.com) 支持 [MDC 语法](https://content.nuxt.com/usage/markdown)：在 Markdown 中无缝集成组件，打造动态内容。
- 自动生成侧边栏导航：根据内容结构自动生成导航。
- 全文搜索：内置搜索，便于发现内容。
- 优化的排版：更佳的可读性与视觉层次。
- 深色模式：支持暗黑主题，贴合用户偏好。
- 功能丰富：探索模板，发掘其全部能力。
